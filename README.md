# Haloween
Landing page “Noche de Halloween”: responsive HTML/CSS website that shows Halloween traditions around the world, decoration ideas, and a costume catalog with an order form.

Noche de Halloween es una mini-web hecha con HTML y CSS puro que incluye una portada con ejemplos de Halloween en distintos países, enlaces a recursos de decoración y una página de disfraces con tabla de productos y formulario de pedido.
